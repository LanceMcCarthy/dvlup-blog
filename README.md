# Dvlup Blog - Wordpress to Hugo Replacement

I exported my original blog using [WordPress to Hugo Exporter](https://github.com/SchumacherFM/wordpress-to-hugo-exporter), this repo contains the source for the site.

- Live Blog => [https://dvlup.com](https://www.dvlup.com) (running ona )

The website is now a Docker image that can be hosted in any environment. The site is now running on a Raspberry Pi 4, behind a reverse proxy that handles the SSL, and is much faster than it has ever been on cloud hosting.
